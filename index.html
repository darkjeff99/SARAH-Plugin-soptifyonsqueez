Le plugin SoptifyOnSqueez permet de lancer des playslites de spotify sur votre squeeze box

Pré-requis :

  • Un compte Spotify
  • Un serveur squeezebox
  • Le plugins Spotify activé sur votre squeezebox
  • Le plugins squeezebox sur SARAH

La premiere action à réaliser est de mettre à jour les playlistsen disant:

  • SARAH mets à jour les playlistes
Sarah va récupérer les playlists et généré les phrase pour lancer les playlists par exemple :
  • SARAH lance la playlist Metallica

Il faut lancer l'update à chaque ajout ou suppression de playliste sur votre spotify.

ATTENTION : SARAH ne gère pas l'anglais. Si vous avez des playlistes avec des noms en anglais changé le nom (et faite un update) ou dite le en français (oui je sais ça fait mal).
